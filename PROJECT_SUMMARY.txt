=====================================
   SEB Agent — סיכום פרויקט
   עדכון אחרון: יוני 2026
=====================================

מה הפרויקט:
סוכן AI שרץ ברקע על מחשב Windows ומאפשר לתלמידות
לתקשר עם AI מתוך דפדפן מאובטח (Safe Exam Browser — SEB),
שחוסם העתקה, הדבקה וצילום מסך.

פלטפורמת הלמידה: Ultracode (ultracode.co.il)
עורך קוד: Monaco editor בתוך Chrome

=====================================
 ארכיטקטורה טכנית
=====================================

קריאת טקסט:
  Windows UIAutomation API — קורא טקסט ישירות מהחלון הפעיל.
  לא נחסם ע"י SEB כי הוא API של מערכת ההפעלה.
  לא משתמש ב-screenshot ולא ב-OCR — מהיר וקל.
  עובד עם Monaco editor של Ultracode.

זיהוי תגיות:
  Regex: @@TAGNAME: content@@ או @@TAGNAME@@
  Debounce 1.5 שניות — שולח רק כשהתלמידה סיימה להקליד.
  Hash — מונע שליחה כפולה של אותה תגית.
  הסוכן מתעלם מחלון ה-Terminal שלו עצמו.

מודל AI:
  Google Gemini 2.0 Flash Lite — חינמי לחלוטין.
  מפתח API מ: https://aistudio.google.com/apikey
  הפורמט החדש של המפתח מתחיל ב-AQ. (לא AIzaSy).
  Retry אוטומטי x3 על שגיאות 429 (rate limit).
  חיסכון בטוקנים: system prompts קצרים, max_tokens מוגבל.

תגובה:
  חלון tkinter always-on-top עם התשובה.
  כפתור "כתוב בעורך" — מקליד לתוך עורך הקוד (עוקף חסימת הדבקה).
  כפתור "העתק" — שם ב-Clipboard.

=====================================
 תגיות זמינות
=====================================

@@ASK: שאלה@@        — שאלה טכנית קצרה (הסבר בלי קוד)
@@ASKALL: שאלה@@     — שאלה על הקוד שמעל התגית
@@SOLVE: משימה@@     — בקשת קוד לפי תיאור
@@SOLVEALL@@         — פיתרון כל התרגיל לפי מה שבשדה
@@FIX: הוראה@@       — תיקון הקוד שמעל התגית
@@COPY: טקסט@@       — שים טקסט ספציפי ב-Clipboard
@@COPYABOVE@@        — שים את הטקסט שמעל ב-Clipboard  [חוזר כל פעם]
@@COPYALL@@          — שים את כל תוכן השדה ב-Clipboard  [חוזר כל פעם]
@@SCREENSHOT@@       — צלם מסך ושמור לקובץ PNG בתיקיית seb_agent  [חוזר כל פעם]

=====================================
 קבצי הפרויקט
=====================================

main.py          — לולאה ראשית, debounce, ניהול תגיות
reader.py        — קריאת טקסט מ-UIAutomation
detector.py      — זיהוי תגיות, hash למניעת כפילויות
claude_client.py — קריאות Gemini API, retry, token efficiency
popup_window.py  — חלון תשובה (tkinter, always-on-top)
typer.py         — הקלדה אוטומטית לתוך עורך הקוד
config.py        — טעינת settings.json
settings.json    — API key + הגדרות
install.bat         — התקנת dependencies (פעם אחת)
start.bat           — הרצת הסוכן
.gitignore          — מונע העלאת settings.json ותמונות ל-GitHub
TAGS_GUIDE.txt      — מדריך שימוש בתגיות
PROJECT_SUMMARY.txt — קובץ זה

חשוב: settings.json לא עולה ל-GitHub (מכיל API key).
כל מחשב של תלמידה צריך settings.json משלה עם ה-API key.

=====================================
 הגדרות (settings.json)
=====================================

api_key        — Gemini API key
model          — gemini-2.0-flash-lite (ברירת מחדל, חינמי)
poll_interval  — שניות בין קריאות (0.3)
response_mode  — "auto" = חלון קופץ | "type" = כתיבה ישירה לעורך
type_speed     — מהירות הקלדה אוטומטית (0.015 שניות לתו)

=====================================
 שימוש יומיומי
=====================================

לפני הבחינה (על מחשב כל תלמידה):
  1. install.bat — פעם אחת בלבד (מתקין)
  2. הכנסת API key ב-settings.json
  3. start.bat — לפני פתיחת SEB

במהלך הבחינה:
  - start.bat רץ ברקע, SEB פתוח רגיל
  - התלמידה כותבת תגית בעורך הקוד
  - מחכה 2 שניות
  - חלון תשובה מופיע

=====================================
 אזהרות חשובות לשימוש
=====================================

- אל תפתחי קבצי טקסט (TAGS_GUIDE.txt וכד') בזמן הבחינה —
  הסוכן עלול לקרוא את ה-@@ שבהם ולהתבלבל
- אל תעלי את settings.json ל-GitHub — הוא מכיל API key פרטי
  (מוגן ע"י .gitignore)
- אם תגית לא עובדת: לוודא שיש @@ בסוף ולחכות 2 שניות
- אם SEB פתוח במצב kiosk מלא וחלון לא מופיע:
  לשנות response_mode ל-"type" ב-settings.json

=====================================
 בעיות ידועות ופתרונות
=====================================

  פתרון: השתמש ב-google-genai (REST) — כבר מוטמע.
  אם עדיין קורה: הרץ install.bat שוב.

429 Too Many Requests:
  פתרון: retry אוטומטי x3 עם המתנה 15 שניות — מוטמע.

הסוכן קורא את ה-Terminal שלו עצמו:
  פתרון: רשימת חלונות מתעלמים — מוטמע.

חלון קופץ נחסם ע"י SEB (kiosk מלא):
  פתרון: שנה response_mode ל-"type" ב-settings.json

תגית לא מזוהה:
  לבדוק: האם יש @@ סגירה? האם חיכית 2 שניות?
